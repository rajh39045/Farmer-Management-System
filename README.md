# 📖 Project Overview

Krishi Market is a complete Farmer-to-Consumer Marketplace where farmers can list fresh agricultural products and customers can purchase them directly without intermediaries.

The platform provides secure authentication, product management, shopping cart, wishlist, order management, reviews, notifications, dashboards, image uploads, and responsive user interfaces.
